# QA Log Analyzer

A bash tool for ingesting and analyzing application log files. Built as part of my QA engineering portfolio.

## What it does
- Ingests a sample application log file
- Extracts and categorizes ERROR, WARNING, and INFO lines
- Counts each log level
- Saves errors to a separate file for triage
- Outputs a clean summary report

## Tech
- Bash scripting
- grep, awk, cat, tail
- Designed for Linux/macOS terminal
